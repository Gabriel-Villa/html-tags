action: Especifica la URL a la que se envían los datos del formulario.
method: Define el método HTTP que se usa al enviar el formulario. Los valores comunes son GET y POST.
enctype: Especifica el tipo de codificación de datos que se usa al enviar el formulario. Los valores comunes son application/x-www-form-urlencoded (por defecto), multipart/form-data (para enviar archivos) y text/plain.
autocomplete: Indica si el navegador debe completar automáticamente los campos del formulario. Los valores pueden ser on o off.
novalidate: Desactiva la validación del formulario en el lado del cliente.
target: Especifica dónde mostrar la respuesta que se recibe después de enviar el formulario. Los valores comunes son _self, _blank, _parent, _top o el nombre de un iframe.
name: Define el nombre del formulario, que se puede usar para identificarlo con JavaScript.
accept-charset: Especifica el conjunto de caracteres que el servidor puede manejar para los datos del formulario.
rel: Define la relación entre el documento actual y el recurso al que apunta el atributo action.
